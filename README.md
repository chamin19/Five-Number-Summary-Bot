# Five-Number-Summary-Bot

The Five-Number Summary is useful for producing a standard boxplot model to depict the distribution of a data set. This is a simple bot that uses Python and the Discord API to perform standard Five-Number Summary calculations. 

## Usage
1. If the bot is online, it will send a greeting and a list of its commands to the channel.
 
   <img src="https://github.com/hanmin19/Five-Number-Summary-Bot/blob/main/demo_img/on_ready.jpg" width="400">
   
   
2. Enter "!new_data" followed by a set of numbers separated by commas or spaces. 

   <img src="https://github.com/hanmin19/Five-Number-Summary-Bot/blob/main/demo_img/new_data.jpg" width="400">
   
   
3. Message the channel with the desired command. The bot will display a sorted list of the data previously entered by the user, followed by the respective calculation. 

Commands | Calculation
---------|------------------------
!A       | Mean, Median, Mode
!B       | Sample Variance, Standard Deviation
!C       | First Quartile, Third Quartile
!D       | Lower Fence, Upper Fence, Outliers
!E       | Absolute Minimum, Absolute Maximum
!F       | Minimum and Maximum within lower and upper fence
